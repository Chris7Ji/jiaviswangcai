# 微信公众号自动发布配置指南

## 📋 配置步骤

### 第一步：获取公众号凭证

1. 登录[微信公众平台](https://mp.weixin.qq.com/)
2. 左侧菜单 → **开发** → **基本配置**
3. 获取以下信息：
   - **AppID(应用ID)**: wx开头的一串字符
   - **AppSecret(应用密钥)**: 点击"重置"获取（注意：只显示一次，请保存）

### 第二步：配置IP白名单

1. 在同一页面，找到"IP白名单"
2. 点击"查看"
3. 添加您的服务器IP地址
   - 如果您不确定IP，可以运行: `curl ifconfig.me`
   - 或者添加 `0.0.0.0/0`（不推荐，有安全风险）

### 第三步：运行配置脚本

```bash
cd ~/.openclaw/workspace
source venv_ppt/bin/activate
python3 wechat_publisher.py
```

按提示输入：
- AppID
- AppSecret

### 第四步：测试连接

脚本会自动测试API连接，如果显示"✅ API连接测试成功"，则配置完成。

---

## 🚀 发布文章

配置完成后，运行发布脚本：

```bash
python3 publish_article.py
```

流程：
1. 加载已保存的配置
2. 读取文章文件
3. 转换Markdown为HTML
4. 创建草稿
5. 您在微信公众平台确认并发布

---

## ⚠️ 注意事项

### 权限要求
- 公众号必须是**认证公众号**（订阅号或服务号）
- 个人订阅号部分接口受限
- 需要开通"素材管理"权限

### 安全提醒
- AppSecret非常重要，不要泄露
- 配置保存在本地 `wechat_config.json`
- 建议设置IP白名单，限制访问来源

### 发布限制
- 每天最多发布8篇图文消息
- 草稿保存后需要在24小时内发布
- 封面图片需要提前上传到微信服务器

## 🔧 故障排查

### 问题1: "获取Token失败"
**原因**: 
- AppID或AppSecret错误
- IP白名单未配置
- 公众号未认证

**解决**:
1. 检查AppID和AppSecret
2. 确认IP白名单已添加
3. 确认公众号已认证

### 问题2: "草稿创建失败"
**原因**:
- 文章内容包含违规内容
- 文章格式不正确
- 超过字数限制（正文最多2万字）

**解决**:
1. 检查文章内容
2. 简化HTML格式
3. 控制文章长度

### 问题3: "图片上传失败"
**原因**:
- 图片格式不支持（只支持JPG/PNG）
- 图片大小超过限制（最大2MB）
- 图片路径错误

**解决**:
1. 转换图片格式
2. 压缩图片大小
3. 检查图片路径

## 💡 高级功能

### 定时发布
微信公众号API不支持直接定时发布，但您可以：
1. 创建草稿
2. 在微信公众平台设置定时发送

### 封面图片
目前脚本不支持自动上传封面图片，需要您：
1. 在微信公众平台编辑草稿时上传
2. 或使用微信素材管理API先上传图片

### 原创声明
发布时可以在微信公众平台勾选"原创声明"。
