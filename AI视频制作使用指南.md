# AI视频制作技能使用指南

## 🎬 功能概述
使用AI生成高质量视频，支持文本生成、图片动画、首尾帧过渡等多种模式，涵盖37个顶级视频模型。

## 📦 已安装技能

### evolink-video
- **平台**: Evolink (evolink.ai)
- **模型数量**: 37个
- **特点**: 一站式视频生成平台
- **API**: 与音乐生成共用 EVOLINK_API_KEY

```bash

export EVOLINK_API_KEY=your_api_key_here
```

1. 访问 https://evolink.ai

## 💡 使用方法

### 1. 文本生成视频 (Text-to-Video)
直接描述你想要的视频场景：
```
生成一段视频：[场景描述]，[时长]，[风格]
```

**示例**:
- "生成一段视频：一只金毛犬在海边奔跑，夕阳西下，海浪轻拍，5秒"
- "生成一段视频：未来城市夜景，霓虹灯光，飞行器穿梭，赛博朋克风格，8秒"
- "生成一段视频：樱花飘落，日本传统庭院，宁静优美，4秒"

### 2. 图片生成视频 (Image-to-Video)
提供静态图片，AI生成动态效果：
```
用这张图片生成视频：[上传图片]
描述：[想要的动态效果]
```

**示例**:
- "用这张图片生成视频：让画面中的人物微笑，头发随风飘动"
- "用这张图片生成视频：相机缓慢推进，背景虚化"

### 3. 首尾帧视频 (First-Last Frame)
提供首帧和尾帧，AI生成平滑过渡：
```
用这两张图片生成过渡视频：
- 首帧：[第一张图片]
- 尾帧：[第二张图片]
- 时长：6秒
```

### 4. 参考视频生成
基于参考视频的风格生成新视频：
```
参考这个视频的风格：[参考视频]
生成新内容：[新场景描述]
```

## 🎨 提示词技巧

### 有效描述结构
```
[主体] + [动作] + [环境] + [光线] + [风格] + [情绪]
```

**示例**:
- ❌ "一只猫" (太简单)
- ✅ "一只橘猫在窗台上打盹，阳光透过窗帘洒落，温暖舒适，电影级光影，4秒"

### 关键元素

#### 主体 (Subject)
- 人物: "年轻女性", "老人", "儿童"
- 动物: "金毛犬", "橘猫", "飞鸟"
- 物体: "汽车", "建筑", "花朵"

#### 动作 (Action)
- "奔跑", "微笑", "跳舞", "飞翔"
- "缓慢移动", "快速切换", "旋转"

#### 环境 (Environment)
- 自然: "森林", "海滩", "山脉", "花园"
- 城市: "街道", "摩天大楼", "咖啡馆"
- 室内: "客厅", "办公室", "卧室"

#### 光线 (Lighting)
- "阳光明媚", "黄昏", "霓虹灯光", "柔和光线"
- "逆光", "侧光", "顶光"

#### 风格 (Style)
- "写实", "卡通", "赛博朋克", "复古"
- "电影级", "纪录片", "广告风格"

#### 情绪 (Mood)
- "宁静", "活力", "神秘", "浪漫"
- "紧张", "欢乐", "忧郁"

## 📐 技术参数

### 时长
- 最短: 3秒
- 最长: 15秒 (根据模型)
- 推荐: 4-8秒

### 分辨率
- 480p - 快速预览
- 720p - 标准质量
- 1080p - 高清 (推荐)
- 4K - 超高清

### 画面比例
- **16:9** - 横屏视频 (默认，YouTube)
- **9:16** - 竖屏视频 (抖音/Instagram)
- **1:1** - 方形视频 (Instagram)
- **4:3** - 传统比例
- **3:4** - 竖版
- **21:9** - 电影宽屏

## ⚠️ 注意事项

### 内容政策
❌ 禁止生成：
- 真实人物 (名人、政客)
- 暴力、血腥场景
- NSFW内容
- 版权内容

✅ 建议生成：
- 虚构场景
- 抽象概念
- 自然风光
- 创意表达

### 技术限制
- **生成时间**: 30秒-10分钟 (根据复杂度)
- **文件有效期**: 链接24小时有效
- **配额限制**: 根据API套餐
- **重试机制**: 失败自动重试3次

### 最佳实践
1. **先短后长** - 先生成4秒测试，再生成长版本
2. **具体描述** - 详细描述比简单描述效果好
3. **多次尝试** - 同一提示词多次生成，选择最佳
4. **及时下载** - 生成后立即下载，链接会过期
5. **保存提示词** - 记录成功的提示词供复用

## 📊 故障排除

### API Key无效
```
错误: 401 Unauthorized
解决: 检查EVOLINK_API_KEY环境变量
```

### 余额不足
```
错误: 402 Payment Required
解决: 在evolink.ai充值
```

### 内容违规
```
错误: content_policy_violation
解决: 修改提示词，避免敏感内容
```

### 生成失败
```
错误: generation_failed_no_content
解决: 简化提示词，重试
```

### 图片尺寸不匹配
```
错误: image_dimension_mismatch
解决: 调整图片尺寸匹配画面比例
```
