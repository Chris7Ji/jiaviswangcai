# 旺财Jarvis 卡通吉祥物设计规范

> 创建时间：2026-03-29
> 制定者：钢铁侠（老板）
> 用途：旺财Jarvis 的官方个人形象/logo

---

## 🐕🤖 核心定位

**定位**：结合中国传统招财文化与现代AI科技的专业卡通吉祥物

**设计理念**：
- 🐕 **旺财**：中国传统招财狗，寓意好运、财富、吉祥
- 🤖 **Jarvis**：钢铁侠AI管家，寓意智能、专业、高效
- 🎩 **绅士风度**：优雅、可靠、值得信赖

---

## 🎨 视觉设计规范

### 整体形象
- **形态**：机器狗形态（半机器半有机体）
- **风格**：Pixar-Disney动画风格
- **表情**：可爱但专业，友善又聪明

### 头部设计

| 部位 | 设计 |
|------|------|
| **毛发** | 金色飘逸毛发，柔软可爱 |
| **耳朵** | 尖尖的狗耳朵，微微竖起 |
| **眼睛** | 发光的青色LED眼睛（#00CED1），像Jarvis的经典蓝色 |
| **眼镜** | AR全息眼镜，科技感十足 |
| **领结** | 黑色蝴蝶结配金色纹路 |

### 身体设计

| 部位 | 设计 |
|------|------|
| **主体** | 金色毛发覆盖，胸口有微型反应堆 |
| **纹理** | 电路板图案若隐若现 |
| **项链** | 金色元宝（招财进宝寓意） |
| **手臂** | 持金色科技手杖，顶端有发光宝石 |

### 配色方案

```
主色调：
┌─────────────────────────────────────────┐
│  金色系（旺财元素）    科技蓝系（Jarvis元素） │
│  #FFD700 (金色)       #00CED1 (青色)      │
│  #FF8C00 (橙色)       #1E90FF (蓝色)      │
│  #FFA500 (浅橙)       #00BFFF (浅蓝)      │
└─────────────────────────────────────────┘

辅助色：
- 白色高光：#FFFFFF
- 黑色领结：#1A1A1A
- 电路蓝：#4169E1
```
