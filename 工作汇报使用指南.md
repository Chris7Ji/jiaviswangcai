# 工作汇报生成技能使用指南

## 🎯 功能概述
自动从Session历史、任务记录生成结构化工作汇报（日报/周报），支持PDF导出和多渠道发送。

## 📦 已安装技能

### 1. daily-report-skill (日报生成器)
- **功能**: 自动生成每日工作总结
- **输出**: Markdown格式，支持PDF转换
- **触发**: 手动或定时自动

### 2. weekly-report (周报生成器)
- **功能**: 自动汇总本周工作
- **输出**: 简洁周报格式
- **触发**: 关键词触发

## 📊 报告内容结构

### 日报模板包含：
1. **📋 工作完成项** - 今日完成的任务列表
2. **📚 学习收获** - 新知识和洞察
3. **🔍 复盘反思**
   - 做得好的地方
   - 需要改进的地方
   - 发现的问题
4. **💡 建议反馈** - 给老板的建设性建议
5. **🧬 自我进化** - 记忆更新、技能变化等

### 周报模板包含：
- 本周工作汇总
- 关键成果
- 下周计划

## 📄 输出格式

### Markdown格式
- 保存位置: `memory/daily-reports/YYYY-MM-DD.md`
- 可直接查看和编辑
- 支持Markdown所有特性

### PDF格式
- 使用 md-to-pdf-cjk 技能转换
- 支持中文字体
- 专业排版

## ⏰ 定时自动生成

可通过OpenClaw cron配置定时任务：

```json
{
  "schedule": {
    "kind": "cron",
    "expr": "0 18 * * *",
    "tz": "Asia/Shanghai"
  },
  "payload": {
    "kind": "agentTurn",
    "message": "generate daily report"
  }
}
```

**建议时间**:
- 日报: 每天 18:00 (下班前)
- 周报: 每周五 17:00 (周末前)

## 🎨 自定义模板

可修改模板文件自定义报告格式：
```
/Users/jiyingguo/.openclaw/workspace/skills/daily-report-skill/assets/template.md
```

**支持的变量**:
- `{{date}}` - 日期 (YYYY-MM-DD)
- `{{time}}` - 时间 (HH:MM)
- `{{recipient}}` - 收件人姓名
- `{{agent_name}}` - Agent名称

## 📱 发送渠道

当前支持:
- ✅ 飞书 (Feishu)
- ✅ Discord
- ✅ Slack
- ✅ Telegram

## ⚠️ 注意事项

1. **数据收集**: 自动从Session历史、memory目录收集工作记录
2. **隐私保护**: 敏感信息会自动过滤
3. **手动补充**: 可手动添加遗漏的工作项
4. **历史查看**: 所有报告保存在 `memory/daily-reports/` 目录

- **md-to-pdf-cjk**: Markdown转PDF
- **summarize**: 内容总结
