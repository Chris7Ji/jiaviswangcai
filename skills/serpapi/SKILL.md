# SerpAPI 联网搜索技能

## 描述
使用SerpAPI进行联网搜索，获取实时搜索结果。

## 配置
需要SerpAPI密钥：https://serpapi.com/

## 使用方法
```bash
# 设置API密钥
export SERPAPI_API_KEY=your_api_key_here

# 使用搜索
serpapi-search "搜索关键词"
```

## 工具
- `serpapi_search`: 执行SerpAPI搜索
- `web_search`: 使用SerpAPI进行网页搜索

## 注意事项
- 需要有效的SerpAPI账户和API密钥
- 有使用限制，请查看SerpAPI定价
- 支持Google、Bing、Yahoo等搜索引擎
