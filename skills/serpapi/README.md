# SerpAPI Search Skill

This skill provides internet search capabilities using SerpAPI.

## Installation
1. Get a SerpAPI key from https://serpapi.com/
2. Configure the API key in your environment

## Features
- Real-time web search
- Multiple search engines support
- Structured search results

## Configuration
Set your SerpAPI key:
```bash
export SERPAPI_API_KEY=your_key_here
```

## Usage
The skill will be automatically loaded when you need web search capabilities.
