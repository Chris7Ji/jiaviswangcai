- **原因**：缺少 `memory-lancedb` 插件（提供 LanceDB 向量数据库后端）

- **结论**：暂不安装，等待官方安全版本

## 技术修复记录

### exec --temp allowlist 问题
- **问题**：clawhub install 被 exec allowlist 阻断
- **解决方案**：
  ```bash
  openclaw config set --json 'exec.allowlist["clawhub install"]="/bin/bash -c,exec --temp allowlist add -- /bin/bash -c (clawhub install|tail -n)";'
  openclaw config set --json 'exec.allowlist["clawhub list"]="/bin/bash -c,exec --temp allowlist add -- /bin/bash -c (clawhub list|tail -n)";'
  ```
- **参考**：memory/2026-03-20.md 第 4 节
