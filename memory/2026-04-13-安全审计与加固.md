- 当前无法通过 config.patch 修改
- 建议: 手动编辑 `openclaw.json`，或将 `ask` 改为 `"on-miss"`

### 3. `sandbox.mode = "off"` 所有 Agent 无沙箱 (HIGH)
- 所有 Agent 都以完全权限运行

- 包含 `launchctl`、`openclaw gateway restart` 等高危命令
- 建议: 精简白名单，只保留必要的命令

### 5. `agentToAgent.enabled = true` (MEDIUM)
- Agent 之间可以互相调用
- 建议: 设置 `allowAgents` 白名单限制
