# 使用秀米/135编辑器发布到微信公众号

## 🎯 推荐工具

### 1. 秀米 XIUMI（推荐）
- 网址: https://xiumi.us/
- 优点: 排版精美，模板丰富，支持Markdown导入
- 缺点: 免费版功能有限

### 2. 135编辑器
- 网址: https://135editor.com/
- 优点: 功能全面，运营工具多
- 缺点: 界面较复杂

---

## 📱 使用秀米发布步骤

### Step 1: 注册/登录
1. 打开 https://xiumi.us/
2. 使用微信扫码登录

### Step 2: 创建图文
1. 点击"新建图文"
2. 选择"导入文章"

### Step 3: 导入文章
**方法一：直接复制粘贴**
1. 打开文件：`wechat_article_xiumi.html`
2. 全选复制（Cmd+A, Cmd+C）
3. 在秀米编辑器中粘贴

**方法二：使用Markdown导入**
1. 打开文件：`wechat_article_openclaw.md`
2. 复制内容
3. 秀米 → 工具 → Markdown导入

### Step 4: 排版优化
1. **设置标题**：调整字号、颜色
2. **调整段落**：行距1.75-2.0
3. **添加重点**：使用秀米的"重点框"、"引用"样式
4. **插入图片**：
   - 封面图：建议使用OpenClaw Logo或GitHub截图
   - 配图：根据内容插入相关图片

### Step 5: 同步到公众号
1. 点击右上角"同步到公众号"
2. 扫码授权
3. 选择要同步的公众号
4. 确认同步

### Step 6: 在公众号后台发布
1. 登录 mp.weixin.qq.com
2. 进入"内容与互动" → "草稿箱"
3. 找到同步过来的文章
4. 设置封面图片（900×383像素）
5. 填写摘要
6. 预览确认
7. 点击"群发"发布

---

## 🎨 排版建议

### 标题样式
- 主标题：18-20px，加粗，深色
- 小标题：16-18px，加粗，可使用品牌色

### 正文样式
- 字号：15-16px
- 行距：1.75-2.0
- 段间距：10-15px

### 重点突出
- 使用秀米的"重点框"组件
- 关键数据用彩色背景
- 金句使用引用样式

### 配色方案
- 主色：深蓝 #1E3A5F（科技、专业）
- 强调色：橙色 #F47C20（活力、重点）
- 文字：深灰 #333333
- 背景：白色或浅灰 #F5F5F5
