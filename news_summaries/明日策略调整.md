# 📰 AI新闻摘要策略调整方案
**生效时间：2026年3月18日起**

## 🎯 **核心原则**
**新闻内容要新，不是文章发布时间新**

## 🔍 **筛选标准**

### ✅ **必须满足**
1. **事件发生时间** - 必须是最近1-2天内发生的真实事件
2. **官方/权威来源** - 必须有官方发布或权威媒体报道
3. **具体时间戳** - 事件有明确的发生时间
4. **多源验证** - 至少2个独立来源确认

### ❌ **必须排除**
1. 预测性、分析性文章
2. 趋势报告、行业分析
3. 旧闻重发（即使文章刚发布）
4. 缺乏具体时间的事件

## 📊 **新闻源优化**

### **一级源（权威实时）**
1. **新华网科技频道** - 实时新闻API
2. **人民网科技** - 官方发布
3. **央视新闻科技** - 权威报道
4. **工信部官网** - 政策发布
5. **科技部官网** - 科研进展

### **二级源（专业媒体）**
1. **机器之心** - 但需验证事件时间
2. **36氪** - 但需验证事件时间
3. **IT之家** - 但需验证事件时间

### **三级源（国际媒体）**
1. **OpenAI官方博客** - 直接抓取
2. **Google AI Blog** - 直接抓取
3. **Nature/Science** - 学术发布

## ⚙️ **技术实现**

### **时间验证机制**
```
1. 提取文章中的事件时间
2. 对比当前时间
3. 只保留最近48小时内的事件
4. 标记时间不明确的文章
```

### **内容质量评分**
- A级：官方发布 + 具体时间 + 多源验证
- B级：权威报道 + 具体时间
- C级：专业媒体报道（需人工审核）
- D级：排除（预测、分析、时间不明）
