## 📝 手工发布步骤

### Step 1: 新建图文消息
1. 登录后，点击左侧菜单 **"内容与互动"**
2. 点击 **"图文消息"**
3. 点击右上角 **"新建图文消息"** 按钮

### Step 2: 填写标题
1. 在标题栏输入：
   ```
   OpenClaw：一个开源项目如何在3个月内斩获30万Star？
   ```

### Step 3: 粘贴正文
1. 打开文件：`wechat_article_manual.txt`
2. 全选复制（Cmd+A, Cmd+C）
3. 在编辑器正文区域粘贴（Cmd+V）

### Step 4: 格式调整（重要！）

#### 4.1 设置小标题格式
对于每个小标题（如"引子：凌晨三点的邮件"）：
1. 选中标题文字
2. 点击工具栏 **"H2"** 或 **"标题"** 格式
3. 或设置为：字号18px，加粗

需要设置的小标题：
- 引子：凌晨三点的邮件
- 起源篇：一个"老炮儿"的二次创业
- 爆火篇：为什么是它？
- 影响篇：一场静悄悄的生产力革命
- 未来篇：AI助手的iPhone时刻
- 结语

#### 4.2 设置重点文字
对于金句和重点：
1. 选中文字
2. 点击工具栏 **"B"** 加粗
3. 或改变颜色为橙色/红色

重点包括：
- "最好的产品，往往源于创始人自己的痛点。"
- "问答式AI是过去，执行式AI才是未来。"
- "AI不是要取代人，而是要取代那些让人烦死的工作。"
- "未来已来，只是分布不均。"
- "那些愿意拥抱变化的人，总是走在前面。"

#### 4.3 设置列表格式
对于数据列表（如GitHub Stars等）：
1. 选中列表内容
2. 点击工具栏 **"有序列表"** 或 **"无序列表"**

#### 4.4 段落间距
1. 选中正文段落
2. 设置行距为 **1.75** 或 **2.0**
3. 段前段后间距设置为 **10-15px**

### Step 5: 上传封面图片
1. 在右侧找到 **"封面和摘要"** 区域
2. 点击 **"+"** 或 **"选择封面"**
3. 上传图片（建议尺寸：900×383像素）
4. 调整图片显示区域

**封面图建议**：
- OpenClaw Logo + GitHub 297k Stars截图
- 或使用科技感强的AI概念图

### Step 6: 填写摘要
1. 在"摘要"栏输入：
   ```
   一个开源AI助手项目，三个月斩获30万Star。它凭什么让开发者疯狂？本文揭秘OpenClaw的崛起之路。
   ```

### Step 7: 预览检查
1. 点击底部 **"预览"** 按钮
2. 输入您的微信号，发送预览
3. 在手机上查看效果
4. 检查：
   - 标题是否正常
   - 段落是否清晰
   - 重点是否突出
   - 图片是否显示

### Step 8: 保存或发布

#### 选项A：保存为草稿（推荐先保存）
1. 点击 **"保存"** 按钮
2. 文章会保存到草稿箱
3. 可以稍后继续编辑

#### 选项B：立即发布
1. 确认无误后，点击 **"群发"** 按钮
2. 扫描二维码确认
3. 选择发布时间：
   - 立即群发
   - 定时群发（推荐工作日晚8-10点）
4. 确认发布

## 🎨 排版优化技巧

### 字体设置
| 元素 | 字号 | 样式 |
|------|------|------|
| 标题 | 18-20px | 加粗 |
| 小标题 | 16-18px | 加粗 |
| 正文 | 15-16px | 常规 |
| 注释 | 14px | 灰色 |

### 颜色搭配
- 主标题：#1a1a1a（深黑）
- 小标题：#2c3e50（深蓝）
- 正文：#333333（深灰）
- 重点：#e74c3c（红色）或 #f47c20（橙色）
- 引用：#666666（中灰）

### 间距设置
- 行距：1.75-2.0
- 段间距：10-15px
- 标题与正文间距：15-20px
