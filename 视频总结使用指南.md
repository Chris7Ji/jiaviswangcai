# 视频总结技能使用指南

## 🎯 功能概述
自动获取YouTube/Bilibili视频字幕，使用AI总结为结构化Markdown文档。

## 📋 使用方法

### 基本用法
直接发送视频链接并说"帮我总结这个视频"：
```
https://www.youtube.com/watch?v=xxx
帮我总结这个视频
```

### 支持的链接格式

#### YouTube
- ✅ `https://www.youtube.com/watch?v=VIDEO_ID`
- ✅ `https://youtu.be/VIDEO_ID`
- ✅ `https://www.youtube.com/shorts/VIDEO_ID`

#### Bilibili
- ✅ `https://www.bilibili.com/video/BVxxx`
- ✅ `https://www.bilibili.com/video/avxxx`
- ✅ `https://b23.tv/xxx`

## 🔧 高级选项

### 总结长度
- `--length short` - 简短总结（适合快速了解）
- `--length medium` - 中等长度（推荐）
- `--length long` - 详细总结（包含更多细节）
- `--length xl` - 超长总结（最详细）

### 提取字幕
```bash
summarize "视频链接" --extract --format md --markdown-mode llm
```

### 提取幻灯片/PPT
```bash
summarize "视频链接" --slides
```

### 包含时间戳
```bash
summarize "视频链接" --timestamps
```

## 💡 最佳实践

1. **优先使用自动模式**
   - 工具会自动选择最佳的字幕获取方式
   
2. **长视频建议**
   - 对于超过1小时的视频，使用 `--length long` 或 `--length xl`
   - 需要完整字幕时使用 `--extract`

3. **技术/教程视频**
   - 使用 `--slides` 提取关键帧
   - 配合 `--timestamps` 标记重要时间点

4. **多语言视频**
   - 自动检测语言
   - 输出中文总结（已配置）

## ⚠️ 注意事项

- 需要视频有字幕或自动生成字幕
- 部分私密视频可能无法获取
- Bilibili视频可能需要额外配置
- 网络问题可能导致获取失败

## 📊 输出格式

生成的Markdown包含：
- 视频标题和基本信息
- 内容摘要（按主题分段）
- 关键要点列表
- 重要时间戳（如启用）
- 相关链接和资源
