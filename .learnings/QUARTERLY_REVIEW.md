# 每季审查检查清单
# 执行频率: 每季度末（3月、6月、9月、12月）
# 执行者: AI助理旺财
# 指示来源: 老板（季）

## 🔍 每季审查任务

### 1. 全面审查MEMORY.md
- [ ] 检查所有记忆条目是否仍然有效
- [ ] 删除过时或不再适用的信息
- [ ] 确保总长度 ≤ 100行
- [ ] 更新最后审查日期

### 2. 全面审查AGENTS.md
- [ ] 验证所有Agent配置是否正确
- [ ] 检查Agent列表是否需要更新
- [ ] 优化任务派发规则
- [ ] 更新Agent能力描述

### 3. 全面审查SOUL.md
- [ ] 确认核心原则是否仍然适用
- [ ] 更新身份定义（如有变化）
- [ ] 优化边界定义
- [ ] 检查Vibe描述是否准确

### 4. 全面审查TOOLS.md
- [ ] 更新工具配置信息
- [ ] 添加新工具的使用技巧
- [ ] 删除不再使用的工具配置
- [ ] 优化常用路径和配置

### 5. 季度总结归档
- [ ] 将本季度所有月度归档合并
- [ ] 生成本季度总体报告
- [ ] 识别长期趋势和模式
- [ ] 制定下季度改进计划

### 6. 记忆系统健康检查
- [ ] 检查记忆文件总大小
- [ ] 验证归档系统正常工作
- [ ] 测试记忆检索功能
- [ ] 优化记忆加载速度
