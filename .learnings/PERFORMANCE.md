## ⚡ 预加载策略

### 会话启动时立即读取（避免延迟）
```python
# 这些文件在每次会话开始时立即读取
must_read_files = [
    "SOUL.md",           # 核心身份（~5KB）
    "MEMORY.md",         # 核心记忆（~2KB）
    ".learnings/INDEX.md" # 关键词索引（~3KB）
]
# 总大小: ~10KB，读取时间: <100ms
```

### 按需读取（延迟加载）
```python
# 这些文件只在需要时读取
lazy_load_files = {
    "TOOLS.md": "需要工具配置时",
    "AGENTS.md": "需要任务派发时",
    "USER.md": "需要了解老板偏好时",
    "HEARTBEAT.md": "收到心跳消息时",
    ".learnings/by-topic/*.md": "需要特定主题时"
}
```

## 🚀 避免慢速操作

### ❌ 避免这些操作（响应慢）
1. **memory_search** - 经常超时300秒
2. **web_search** - 需要API key，经常失败
3. **大文件读取** - >100KB的文件
4. **递归目录扫描** - 大量文件遍历

### ✅ 使用这些替代方案（响应快）
1. **直接文件读取** - <100ms
2. **关键词索引查找** - INDEX.md
3. **分类文件读取** - by-topic/*.md
4. **缓存结果** - 已读文件内容缓存

## 📊 响应时间优化

### 优化前（慢）
```
用户提问 → memory_search(300s超时) → 失败 → 手动查找 → 回答
总时间: 300秒+（超时后手动处理）
```

### 优化后（快）
```
用户提问 → 查INDEX.md(50ms) → 读对应文件(100ms) → 回答
总时间: <3秒
```
